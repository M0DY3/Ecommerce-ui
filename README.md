## ✨ Features

- 📦 Category icons (Laptop, Phone, TV, Bike, etc.)
- 🛍️ Best-selling product grid with details
- 🎨 Color & size selection for products
- 🛒 Add to Cart functionality with live cart page
- 💡 Minimal, responsive, and smooth UI
